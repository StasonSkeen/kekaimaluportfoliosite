Coding a website that holds resources for learning the Hawaiian Language

type

Code Development

coding a base site

‘Ōlelo Hawai‘i (Hawaiian Lanuguage) is a langauge on the rebound from near extinction. Thanks to the internet finding resources for learning the language has become easier. Although I'm not a fluent speaker I would like to become one and I created this site to share with others. This base site was my first introduction to coding, using HTML to add content, CSS to style the webpage, & JavaScript to have functionality.
